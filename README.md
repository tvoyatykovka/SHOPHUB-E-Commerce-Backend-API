# ShopHub — мой первый серьёзный API на Python

Честно говоря, я всегда думала, что бэкенд — это что-то сложное и не для меня. Но тут решила попробовать и вот что получилось. Это мой первый полноценный проект на FastAPI.

## Что это вообще

Это бэкенд для интернет-магазина. Можно регистрироваться, входить, смотреть товары, собирать корзину и оформлять заказы. Всё как в жизни, только без оплаты (пока!).

## Что я тут сделала

- 🔐 **Регистрация и вход** — с JWT токенами, всё как у взрослых
- 🛍️ **Товары** — можно посмотреть список, отфильтровать по категории
- 🛒 **Корзина** — добавлять, менять количество, удалять
- 📦 **Заказы** — оформлять и смотреть историю
- 📚 **Swagger** — автоматическая документация, очень удобно

## Технологии, которые я использовала

Этот список я составляла, когда выбирала, на чём писать. В итоге взяла то, что сейчас в тренде:

| Штука | Зачем |
|-------|-------|
| Python 3.11 | Один из любимых языков |
| FastAPI | Быстрый и понятный фреймворк |
| PostgreSQL | Настоящая база данных |
| SQLAlchemy | Работа с БД через код |
| JWT | Безопасная авторизация |
| Docker | Чтобы запускать одной командой |
